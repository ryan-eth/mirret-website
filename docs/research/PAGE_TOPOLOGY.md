# Page Topology: Mirret Surface Monitor

Page height: ~4693px at 1440px viewport. Dark theme throughout.

## Sections (top to bottom)

### 1. Navbar (fixed)
- **Position:** fixed, z-index high, overlays all content
- **Interaction:** static
- **Content:** Mirret shield logo, "Platform" dropdown, "Company" dropdown, "Client Portal" link, "Get a Quote" CTA button

### 2. Hero Section
- **Content:** "Surface Monitor" gradient label, H1 heading, body text, 3 feature pills (Detection, Accuracy, Enforcement), hero canvas mockup image with cool cyan video glow
- **Interaction:** Static

### 3. Threat Scanner Section
- **Label:** "THREAT SCANNER" / "AI-NATIVE ANALYSIS"
- **Heading:** "Get full visibility of your attack surface in minutes"
- **Layout:** 2 cards side by side
  - Card 1: "Effortless onboarding" - brand scan configuration UI mockup
  - Card 2: "Adaptive monitoring" - Sentinel AI chat UI mockup
- **Interaction:** Static

### 4. Detection & Results Section
- **Heading:** "See exactly what threats are targeting your brand"
- **Layout:** 2 cards side by side
  - Card 1: "Precise detection" (Detection Rules) - code editor UI mockup
  - Card 2: "Real-time results" (Scan Results) - scan results dashboard mockup
- **Interaction:** Static

### 5. Timeline Section
- **Label:** "HOW IT WORKS"
- **Heading:** "From zero to full coverage in days"
- **Layout:** 3 timeline items (Day 1, Day 3, Day 7) with descriptions
- **Interaction:** Static

### 6. CTA Section
- **Label:** "SEE YOUR EXPOSURE"
- **Heading:** "Ready to see what's impersonating your brand right now?"
- **Subtext + button:** "Generate a free brand exposure report..." + "Generate Free Report"
- **Background:** Large "Mirret" text watermark

### 7. Footer
- **Layout:** Mirret shield logo, compliance badges, 3 link columns (Product, Company, Resources), copyright, social links (X, LinkedIn)
- **Interaction:** Hover states on links

## Key Design Tokens
- **Background:** #0f0e0d (near black)
- **Card backgrounds:** Semi-transparent gradients
- **Text:** White with opacity variants (0.87, 0.61, 0.49, 0.45)
- **Accent:** #06b6d4 (cyan)
- **Muted:** #8d8d8d
- **Heading font:** Emilio Light
- **Body font:** Inter
- **Code font:** Geist Mono
